解决Scrapy安装时报错：ImportError: No module named twisted.persisted


2017-5-14 Debug Python Scrapy Twisted

出错原因不清楚．用pip install scrapy，在使用的时侯就会报这样的错．然后用pip install twisted，会出现找不到相关的twisted包的问题．一番Google之后，终于找到解决方法

环境：

Ubuntu16.04LTS, Python3.6

方法一：

使用 conda 安装 Scrapy:

conda install scrapy

方法二:

通过tar包安装 twisted

wget https://pypi.python.org/packages/c0/7c/c1e5b61e30b7ffc96576d2a922615c8068e6996a622be813fc626cef07aa/Twisted-16.3.0.tar.bz2#md5=e044af844623e9fbcbe29f578db6053a

tar jxvf Twisted-16.3.0.tar.bz2

cd Twisted-16.3.0

python3 setup.py install
